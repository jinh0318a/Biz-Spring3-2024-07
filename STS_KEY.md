# sts 바로가기 key
- `코드 한줄 삭제하기` : Ctrl + d
- `한줄 복사하기` : 블럭, Ctrl + c
- `붙여 넣기` : Ctrl + v
- `한줄 잘라내기` : 블럭, Ctrl + x
- `키보드로 블럭 설정` : Shilft + 방향키
- `코드 라인의 맨앞으로 가기` : Home
- `코드 라인의 맨끝으로 가기` : End
- `코드 전체 맨 마지막 라인으로 가기` : Ctrl + End, Ctrl + PgDn
- `코드 전체 맨 처음 라인으로 가기` : Ctrl + Home, Ctrl + PgUp
- `코드 생성 Menu 호출` : Alt + Shift + s
- `import 명령라인 최적화` : Ctrl + Shift + o
- `파일, 폴더 이름 변경` : F2
- `새로고침` : F5
- `저장하기` : Ctrl + s
